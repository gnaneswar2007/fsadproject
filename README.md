# FoodSecurity

A full-stack web application that connects surplus food donors with recipients, reducing food waste and improving food security.

## GitHub Repository

**URL**: https://github.com/Ganeshmamidala/Food-Security-project

## Tech Stack

- **React** + **Vite** — frontend framework
- **Tailwind CSS** + **shadcn/ui** — styling and components
- **React Router** — client-side routing

## Roles

| Role | Description |
|------|-------------|
| Admin | Full platform control — manage users, view all donations |
| Donor | List surplus food donations |
| Recipient | Browse and claim available food donations |
| Analyst | View analytics, charts, and impact reports |

## Getting Started

```sh
# Clone the repository
git clone https://github.com/Ganeshmamidala/Food-Security-project.git

# Navigate to project directory
cd Food-Security-project

# Install dependencies
npm install

# Start development server
npm run dev
```

## Features

- Role-based dashboards (Admin, Donor, Recipient, Analyst)
- Real-time donation listings with category filters
- Analytics with charts and impact stats
- User management (edit roles, delete users) — Admin only
